My experiments with Python

Updated August 26, 2025

This page showcases my hands-on experiments with tools and concepts I'm currently exploring. I'll continue updating it as I learn new techniques. Right now, you’ll find:

  • Interactive demos that pull real-time data from various sources.
  • A small MicroPython program for Raspberry Pi Pico that collects and displays environmental data using sensors.
  • A simple web app that flips a virtual coin to show "Yes" or "No", complete with animation and sound — useful for quick decisions.
  • A browser-based tool that converts Ukrainian Cyrillic text into Latin script (CMU 2010, DSTU 9112:2021)
  • A bot that fetches weather data from the Deutscher Wetterdienst, formats it into readable messages, and sends them to Telegram.
  • A small program that draws random Tarot Cards for you and explains their meaning.
  • Curated lists on a range of topics I'm researching.
